# SHGoBelieveIM


一个开源的IM  聊天室功能，单例操作，更简洁的方式接入，集成方便，可使用源码、framework两种方式接入。
同时涵盖了，修改源码，自动打包framewok等一键操作。喜欢的话，请给个star！！！

IM_room   based on  [im_ios](https://github.com/GoBelieveIO/im_ios)
